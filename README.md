Aplikasi administrasi sederhana berbasis Java Swing untuk manajemen pendaftaran jasa cuci sepatu.

Fitur
  
  -Input Data Layanan: Mencatat Kode Layanan, Berat, dan Jam Pengambilan.
 
  -Validasi Kode Layanan: Menggunakan Regex untuk format CSM-XXX-00 (Contoh: CSM-DEP-02).
  
  -Validasi Berat Sepatu: Memastikan input angka desimal presisi (X.XX) tanpa pembulatan.
  
  -Validasi Jam Pengambilan: Memastikan format waktu 24 jam (HH:MM) yang valid.
  
  -Feedback Status: Menampilkan status VALID atau TIDAK VALID secara real-time beserta penjelasan kesalahannya.

#Teknologi
  
  -Java
  
  -Swing GUI (Java Desktop)
  
  -Regular Expression (Regex)

Cara Menjalankan
  
  1.Pastikan Java (JDK) sudah terinstal di perangkat kamu.
  
  2.Compile file CuciSepatuManager.java.
  
  3.Jalankan program melalui terminal atau IDE .
